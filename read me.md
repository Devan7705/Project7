# 🧰 Multi-Utility Toolkit (Python)

A **Python-based Modular Toolkit** that demonstrates the use of **built-in modules, custom modules, and packages**.
This project integrates utilities for **date & time operations, mathematical calculations, random data generation, UUID generation, file operations, and dynamic module exploration**.

---

## 📌 Project Overview

The **Multi-Utility Toolkit** showcases Python’s modular programming features by organizing code into **packages and modules**.

It provides a **menu-driven interface** that allows users to perform multiple utility tasks from a single program.

The project demonstrates the usage of the following Python concepts:

- Built-in modules (`datetime`, `time`, `math`, `random`, `uuid`)
- Custom modules
- Python packages (`__init__.py`)
- `match-case` control structure
- File handling
- `dir()` for dynamic module exploration
- `__name__ == "__main__"` execution control
